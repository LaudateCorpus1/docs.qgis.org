Documentació de QGIS

QGIS té molta documentació. Tota la documentació està en anglès però alguns documents, com la guia d'usuari, també estan disponibles en altres idiomes.

You will find documentation for every QGIS long term release on the respective documentation website.

Get involved and help us write a better documentation.

DOCUMENTACIÓ

  • QGIS 3.4
  • QGIS testing (>3.4)
  • Documentation for older versions

QGIS 3.4

For users:

  • Llegiu el Guia d'usuari QGIS (o en PDF)
  • Seguiu els seminaris en el Manual d'aprenentatge QGIS (o en PDF)
  • Apreneu els fonaments GIS a Una introducció bàsica al GIS

For documentation writers:

  • Apreneu com escriure documentació utilitzant la Guia de Documentació

For developers:

  • Go Python, estudia el Receptari PyQGIS (de connectors i scripts) (o en PDF)
  • Documentació de l'API C++
  • PyQGIS - QGIS Python Api documentation

QGIS de prova

We are still updating (not translating yet) the documentation for releases newer than QGIS 3.4. We call this version 'QGIS testing' and can be found here: http://docs.qgis.org/testing
