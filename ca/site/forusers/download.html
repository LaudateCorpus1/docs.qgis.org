Baixeu QGIS per la vostra plataforma

Binary packages (installers) are available from this page.

La versió actual és QGIS 3.8.2 'Zanzibar' i fou alliberada el 16.08.2019.

The long-term repositories currently offer QGIS 3.4.11 'Madeira'.

Es pot disposar de QGIS a Windows, MacOS X, Linux i Android.

  • Tots els lliuraments

Baixada per a Windows

QGIS in OSGeo4W:
  • OSGeo4W Network Installer (64 bit)
  • OSGeo4W Network Installer (32 bit)

En el instal·lador tria Instal·lació ràpida d'escriptori i selecciona QGIS to install the latest release.
To get the long term release (that is not also the latest release) choose Instal·lació avançada i selecciona qgis-ltr-full
To get the bleeding-edge development build choose Instal·lació avançada i selecciona qgis-dev-full

Standalone installers from OSGeo4W packages
Latest release (richest on features):
  • Versió d'instal·lació independent QGIS 3.8 (64 bit)
  • sha256
  • Versió d'instal·lació independent QGIS 3.8 (32 bit)
  • sha256
Repositori lliurament llarg termini (el més estable):
  • Versió d'instal·lació independent QGIS 3.4 (64 bit)
  • sha256
  • Versió d'instal·lació independent QGIS 3.4 (32 bit)
  • sha256

Baixada per Mac OS X

Official All-in-one, signed installers

Mac Installer Packages for macOS High Sierra (10.13) and newer.

Latest release (richest on features):
  • QGIS macOS Installer Version 3.8
Long term release (most stable):
  • QGIS macOS Installer Version 3.4

Alternative build

Mac Installer Packages for macOS El Capitan (10.11) and newer.

Installation instructions are in the Read Me on the disk image. These packages use the python.org Python 3, version 3.6, the "macosx10.9" build - other distributions are not supported. Install Python before installing QGIS.

Additional GDAL format plugins and PROJ grids are available at kyngchaos.com.
