Material de capacitación¶

En esta página de directorio, usted puede aparecer en la lista si proporciona material de capacitación Libre para QGIS.

La idea general es poder (re)utilizar material de unos y otros.

A los educadores se les anima a utilizar/combinar/añadir material para el Manual de Capacitación de QGIS.

Por favor escriba a psc@qgis.org si quieres agregar algo aquí. O puedes editar esta página en Github y haz un pull request.

Inglés¶

El Manual de Capacitación oficial se puede encontrar aquí: http://docs.qgis.org/latest/en/docs/training_manual/index.html

El material de tutorial, originalmente desarrollado como parte de un taller para el Programa Ecología y Biología Evolutiva en la Universidad de Texas A&M (producido y mantenido por Mike Treglia): http://mltconsecol.github.io/QGIS-Tutorial/

Material de formación presencial, originalmente desarrollado por el Departamento de Transporte de la Ciudad de Nueva York por Datapolitan:

  • `Introducción a los Fundamentos del SIG<http://training.datapolitan.com/qgis-training/Introduction_to_GIS_Fundamentals>`_
  • SIG intermedio con QGIS y PostGIS
  • Código fuente para materiales de capacitación

Portugués de Brasil¶

Roberto Ilacqua ha escrito un manual para clasificación de imágenes en QGIS: Manual do QGIS para Classificação Supervisionada de Áreas, version completa, 200 p - 2017 o Manual do QGIS para Classificação Supervisionada de Áreas (short version), versión corta, 34 p - 2017.

Francés¶

El Ministerio de Ambiente Francés apoya el proyecto QGIS. Su material de capacitación esta disponible aquí: http://www.geoinformations.developpement-durable.gouv.fr/qgis-supports-pedagogiques-r947.html

Otros recursos de ellos están disponibles aquí: http://www.geoinformations.developpement-durable.gouv.fr/qgis-r625.html

El French National Center for Scientific Research (CNRS), ha creado un tutorial web de QGIS: http://ouvrir.passages.cnrs.fr/tutoqgis/

Alemán¶

Material HSR: http://giswiki.hsr.ch/QGIS-Materialien

Checo¶

QGIS para principiantes: http://training.gismentors.eu/qgis-zacatecnik/
