QGIS käyttöohjeet

QGIS ohjelmistolla on paljon dokumentointia. Kaikki dokumentaatio on englanniksi mutta joitakin, kuten Käyttäjän opas, on käännetty muillekin kielille.

You will find documentation for every QGIS long term release on the respective documentation website.

Get involved and help us write a better documentation.
