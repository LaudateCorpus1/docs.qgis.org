Osallistu QGIS yhteisön toimintaan

QGISin on kehittänyt joukkon omistautuneita vapaaehtoisia ja järjestöjä. QGIS pyrkii olemaan vastaanottavainen yhteisö kaikille ihmisille rodusta, uskonnosta, sukupuolesta tai elämänalasta riippumatta.

Myös sinä voit osallistua!

Lisätiedon saamiseksi, ole ystävällinen ja lue moninaisuuden selvitys ja käytännesäännöt.

On useita

  • QGIS postituslistoja,
  • IRC keskusteluja,
  • foorumeita, ja
  • käyttäjäryhmiä
, joita voit käyttää pyytääksesi apua muilta käyttäjilta tai kehittäjiltä, tai vain seurataksesi projektin etenemistä ja tapahtumia. Olet tervetullut liittymään kaikille listoille. Document QGIS

The QGIS project aims to provide clear documentation concerning a User guide, GIS concepts, and training material. Writing documentation provides an ideal platform for your first contribution to the project. If you want to join to the documentation effort, you can find a gentle introduction and guidelines here:

Write Documentation

Käännä QGIS

QGIS hanke tarjoaa mukavan GIS johdannon, käyttäjän oppaan, koodauksen ja kääntämisen oppaan ja API dokumentoinin englanniksi ja jotka joissain tapauksissa on käännetty useille vieraille kielille. Jos olet kiinnostunut pääsemään mukaan dokumentointiponnistuksiin, esimerkiksi kääntämään englanninkielista manuaalia toiselle kielelle, voit oppia kuinka päästä alkuun täältä:

Käännä QGIS kielellesi

Kehitä QGIS ydintä

QGIS ydin muodostuu yhtäältä kirjastoista jotka mahdollistavat Sinun rakentavat räätälöityjä sovelluksia tarjoamalla tehokkaan API:n. Toisaalta se sisältää työasema ja palvelin sovellukset jotka hyödyntävät kirjastojen mahdollisuudet tarjoten samalla käyttäjäystävällisen rajapinnan.
