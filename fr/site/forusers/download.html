Téléchargez QGIS

Les progiciels (installeurs) sont disponibles à partir de cette page.

La version actuelle est QGIS 3.8.2 'Zanzibar', sortie le 16.08.2019.

Les répertoires de long terme offrent couramment QGIS 3.4.11 'Madeira' :

QGIS est disponible sur Windows, MacOS X, Linux et Android

Téléchargement pour Windows

QGIS dans OSGeo4W :
  • Installateur réseau OsGeo4W (64 bit)
  • Installateur réseau OsGeo4W (32 bit)

Dans l'installateur, choisissez Desktop Express Install et sélectionnez QGIS pour installer la dernière version.
Pour obtenir la version long terme (qui peut être différente de la dernière version) choisissez l'Installateur pour utilisateurs experts et sélectionnez qgis-ltr-full
Pour obtenir la version des tout derniers développements choisissez l'Installateur pour utilisateurs experts et sélectionnez qgis-dev-full

Téléchargement pour Mac OS X

Official All-in-one, signed installers

Mac Installer Packages for macOS High Sierra (10.13) and newer.

Dernière version (plus riche en fonctionnalités) :
  • Installeur QGIS pour macOS version 3.8
Version long terme (la plus stable) :
  • Installeur QGIS pour macOS version 3.4

Alternative build

Paquets d'Installation Mac pour macOS El Capitain (10.11) et plus récent.

Les instructions d'installation sont dans le Read Me sur le disque image. Ces paquets utilisent python.org Python 3, version 3.6, "macosx10.9" build - les autres distributions ne sont pas supportées. Installez Python avant d'installer QGIS.

Des extensions supplémentaires de format pour GDAL et des grilles additionnelles pour PROJ sont disponibles à kyngchaos.com.
