Instaladores QGIS¶

Como conseguir QGIS

QGIS está dispoñible gratuitamente en Windows, Linux, MacOS X, BSD e Android.

Recomendamos instalar os paquetes do software publicado.

Para avaliar a inminente versión e permitirlle ós non-desenvolvedores contribuír no desenvolvemento fornecendo paquetes experimentais (olla a advertencia).

Na fase de parada de desenvolvemento que precede ó lanzamento (ver Release schedule) estes paquetes son efectivamente pre-lanzamentos, ós cales urximos ós usuarios para probar.

Windows¶

Para os instaladores de Windows diríxete á Páxina principal dos instaladores.

Para os novos usuarios recomendamos os instaladores autónomos

Para usuarios máis avanzados de QGIS podes usar os paquetes de OSGeo4W, que permiten instalar varias versións en paralelo.

O repositorio OSGeo4W contén un montón de software dos proxectos OSGeo. QGIS e tódalas dependencias están incluídas, xunto con Python, GRASS, GDAL, etc. O instalador é capaz de instalar dende internet ou dende tódolos paquetes necesarios descargados previamente. Os ficheiros descargados gárdanse nun directorio local para futuras instalacións. Os pasos son:

  • Download OSGeo4W Installer (32 bit or 64 bit) and start it

  • Selecciona Instalación Avanzada,

  • navegar á sección Escritorio

  • e coller unha ou máis dos seguintes paquetes:

    Versión Versión Paquete Descrición
    Última Versión 3.8.x Zanzibar ​ qgis Versión
    qgis-rel-dev [1] Compilación automática da próxima versión intermedia
    Repositorio da Versión de Longo Prazo 3.4.x Madeira LTR qgis-ltr Versión
    qgis-ltr-dev [2] Compilación automática da próxima versión de longo prazo intermedia
    Desenvolvemento 3.9 máster qgis-dev [3] Compilación automática da versión de desenvolvemento

Para usuarios que non poden utilizar OSGeo4W por algunha razón ou porque o prefiren hai tamén unha copia semanal de qgis-dev de OSGeo4W como instalador autónomo (tomado o luns). Na fase de parada de desenvolvemento esta actúa como versión candidata.

Ten en conta a advertencia.

Note

Non fagas a cegas unha instalación completa de OSGeo4W. Instala só QGIS e quizais outras compoñentes que queiras. As dependencias incluiranse automaticamente. Unha instalación completa tira de compoñentes que precisan engadidos de terceiras partes, que necesitan ser instalados manualmente. Estes compoñentes fan a instalación inutilizable sen estes engadidos.

Linux¶

A maioría das distribucións linux dividen QGIS en varios paquetes; probablemente precisarás qgis e qgis-python (para executar os plugins). Paquetes como qgis-grass (ou qgis-plugin-grass), qgis-server poden saltarse inicialmente ou instalarse só cando os necesites.

Abaixo podes atopar instrucións para instalar por distribución. Para a maioría das distribucións hai instrucións para QGIS estable e instrucións para instalar innovadoras compilacións experimentais de QGIS (mira a advertencia).

Debian/Ubuntu¶

Os repositorios de software predeterminados de Debian e Ubuntu manteñen frecuentemente versións vellas de QGIS.

Para ter novas versións, tes que engadir repositorios de software alternativos, engadindo unha das liñas deb- de abaixo ó teu ficheiro /etc/apt/sources.list.

O repositorio principal contén múltiples liñas de paquetes de varias versións de Debian e Ubuntu baseadas nas dependencias que as distribucións individuais fornecen.

For Ubuntu we also have extra packages in a separate repository that are based on ubuntugis, which holds more uptodate versions of other GIS packages than Ubuntu itself. If you want those you also need to include ubuntugis-unstable ppa in your /etc/apt/sources.list file (see ubuntugis documentation).

Note

Os paquetes da versión só se producen unha vez xusto despois de que se lance unha nova versión. Xa que non é estable, as versións de debian aínda non lanzadas (de proba) e ubuntugis-unstable poden ter cambios na biblioteca, os paquetes poderían rachar tarde ou cedo para estes obxectivos, cando o desenvolvemento en debian, ubuntu ou ubuntu-unstable cambie e os paquetes usados como dependencias en QGIS cambien. Neste caso tamén podes

  • cambiar a unha versión de debian estable e utilizar os paquetes lanzados con ela,
  • agardar pola próxima versión intermedia (programada para 2019-09-13 12:00:00 UTC), que acontece cada mes e incluirá as dependencias actualizadas,
  • cambiar a un repositorio de desenvolvemento (dispoñible para as dúas ramas da versión e máster) no que os paquetes compílanse de novo de forma regular e tamén recollen automaticamente as dependencias actualizadas ou
  • compilar o teu propio conxunto de paquetes (ver INSTALL).

Engade as liñas para un dos repositorios a /etc/apt/sources.list:

deb     *repository* *codename* main
deb-src *repository* *codename* main

Example latest release for Debian unstable:

deb     https://qgis.org/debian unstable main
deb-src https://qgis.org/debian unstable main

Logo diso teclea as instrucións de abaixo para instalar QGIS:

sudo apt-get update
sudo apt-get install qgis qgis-plugin-grass

In case you would like to install QGIS Server, type:

sudo apt-get update
sudo apt-get install qgis-server

Note

Elimina tódolos paquetes de QGIS e GRASS que teñas instalados doutros repositorios antes de realizar a actualización.

En caso de erros de clave de servidor engade a clave pública do repositorio de qgis.org ó teu chaveiro de paquetes, teclea:

wget -O - https://qgis.org/downloads/qgis-2019.gpg.key | gpg --import
gpg --fingerprint 51F523511C7028C3

Debe resultar:

pub   rsa4096 2019-08-08 [SCEA] [expires: 2020-08-08]
      8D5A 5B20 3548 E500 4487  DD19 51F5 2351 1C70 28C3
uid           [unknown] QGIS Archive Automatic Signing Key (2019) <qgis-developer@lists.osgeo.org>

Logo de verificar a pegada dixital podes engadir a clave ó apt con:

gpg --export --armor 51F523511C7028C3 | sudo apt-key add -

Podes descargar alternativamente a clave dende un servidor de claves e engadila ó apt dunha soa vez (sen verificación manual da pegada dixital):

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-key 51F523511C7028C3

Fedora¶

Consegue os paquetes para calquera Fedora tecleando:

sudo dnf install qgis python3-qgis qgis-grass qgis-server

Os repositorios de software predeterminados de Fedora soen manter versións máis antigas de QGIS.

Para ter versións máis novas, tes que engadir repositorios de software alternativos baseados na versión que queiras instalar (estable, LTR ou de proba).

QGIS estable¶

Activar o repositorio:

sudo dnf copr enable dani/qgis

Logo diso teclea as instrucións de abaixo para instalar QGIS:

sudo dnf install qgis python3-qgis qgis-grass

En caso de que queiras instalar QGIS Server (ten en conta que non é unha práctica habitual instalar ámbalas aplicacións, cliente e servidor, na mesma máquina), teclee:

sudo dnf install qgis-server python3-qgis

This repository also provides a copy of SAGA 2.3.1 compatible with Processing. It can be installed with the following commands:

sudo dnf install saga

openSUSE¶

QGIS estable¶

O último paquete openSUSE estable chamado qgis está dispoñible para 13.1, 13.2, Leap_42.1, Leap_42.2 e Tumbleweed (32 e 64 bits). Engade o seguinte repositorio ó xestor de instalación, onde <VERSION> é por exemplo “openSUSE_Tumbleweed”.

https://download.opensuse.org/repositories/Application:/Geo/<VERSION>/

Tódolos paquetes inclúen compatibilidade con GRASS e Python.

QGIS LTR (Versión de Longo Prazo)¶

O paquete da Versión de Longo Prazo para openSUSE chamado qgis-ltr está dispoñible para 13.1, 13.2, Leap_42.1, Leap_42.2 e Tumbleweed (32 e 64 bits). Engade o seguinte repositorio ó xestor de instalación, onde <VERSION> é por exemplo “openSUSE_Tumbleweed”.

https://download.opensuse.org/repositories/Application:/Geo/<VERSION>/

Tódolos paquetes inclúen compatibilidade con GRASS e Python.

QGIS experimental¶

Un paquete de desenvolvemento actualizado regularmente dende qgis master chamado qgis-master está dispoñible para 13.1, 13.2, Leap_42.1, Leap_42.2 e Tumbleweed (32 e 64 bits). Engade o seguinte repositorio ó xestor de instalación, onde <VERSION> é por exemplo “openSUSE_Tumbleweed”.

https://download.opensuse.org/repositories/Application:/Geo/<VERSION>/

ArchLinux¶

QGIS estable¶

Para Archlinux está dispoñible no respositorio oficial : https://www.archlinux.org/packages/community/x86_64/qgis/

A Versión de Longo Prazo de QGIS está dispoñible en AUR (Repositorio do Usuario Arch).

Para erros e outros comportamentos, ler os comentarios aquí : https://aur.archlinux.org/packages/qgis-git

Mac OS X / macOS¶

As instrucións de instalación están no ficheiro Ler na imaxe do disco. As descargas fanse dende a páxina de descargas de QGIS.

QGIS actual¶

The current QGIS package uses the python.org Python 3.6, at least version 3.6.5, the «macosx10.9» build - other distributions are not supported. Install packages in the numbered order. Especially note that Python must be installed before the GDAL Complete package and QGIS, else the GDAL and other needed Python modules will not be installed.

QGIS estable¶

The stable package uses the system Python 2.7 - other distributions are not supported. It also requires the NumPy and Matplotlib packages on the disk image. Install packages in the numbered order. If an older major version is being upgraded (2.16 or older), delete QGIS.app from your Applications folder before installing this version.

Other Python modules for plugins to use are available from kyngchaos.com.

Instalador Autónomo¶

Hai dispoñible un instalador autónomo en https://lutraconsulting.github.io/qgis-mac-packager/.

FreeBSD¶

QGIS estable¶

To install QGIS from binary packages type

pkg install qgis

Or to customize compilation options, you can build it from FreeBSD ports

cd /usr/ports/graphics/qgis
make install clean

QGIS LTR¶

To install QGIS from binary packages type

pkg install qgis-ltr

Or to customize compilation options, you can build it from FreeBSD ports

cd /usr/ports/graphics/qgis-ltr
make install clean

Flatpak¶

Hai dispoñible un QGIS flatpak experimental para QGIS Estable.

For general Linux Flatpak install notes, see https://flatpak.org/setup/

QGIS on Flathub: https://flathub.org/apps/details/org.qgis.qgis

Para instalar:

flatpak install --from  https://flathub.org/repo/appstream/org.qgis.qgis.flatpakref

Logo executar:

flatpak run org.qgis.qgis

Para actualizar QGIS flatpak:

flatpak update

Ficheiros Flathub: https://github.com/flathub/org.qgis.qgis

Android¶

Hai unha versión experimental dispoñible na tenda google play.

https://play.google.com/store/apps/details?id=org.qgis.qgis

Warning

Actualmente non hai respaldo para Android 5. O mellor respaldo dáse para Android 4.3 e 4.4.x. Esta é un traslado directo da aplicación de escritorio de QGIS. Só está lixeiramente optimizada para dispositivos táctiles e polo tanto precisa avaliarse coidadosamente para a súa idoneidade na utilización día a día. Hai outras aplicacións dispoñibles que están deseñadas e especificamente optimizadas para dispositivos táctiles.

Advertencia de QGIS Experimental¶

Warning

Subminístranse paquetes experimentais de QGIS para varias plataformas ademais da versión estable de QGIS. QGIS experimental contén software non lanzado no que se está a traballar actualmente. Subminístrase só para propósitos de proba para que os adoptantes comproben se os erros foron resoltos e que non fose introducido ningún erro novo. Aínda que tentamos coidadosamente evitar as rupturas, poden en calquera momento non funcionar ou causar estrados nos teus datos. Ten coidado. Fuches advertido!
