Descarga QGIS á túa plataforma

Hai dispoñibles paquetes binarios (instaladores) nesta páxina.

A versión actual é QGIS 3.8.2 'Zanzibar' que foi lanzada en 16.08.2019.

Os repositorios de longo prazo actualmente ofrecen QGIS 3.4.11 'Madeira'.

QGIS está dispoñible en Windows, MacOS X, Linux e Android.

Descarga para Windows

QGIS en OSGeo4W:
  • Instalador en Rede OSGeo4W (64 bit)
  • Instalador en Rede OSGeo4W (32 bit)

No instalador escoller Instalador de Escritorio Exprés e selecciona QGIS para instalar a versión máis recente.
Para conseguir a versión de longo prazo (que non é tampouco a última versión) escolleInstalación Avanzada e selecciona qgis-ltr-full
Para obter a compilación de desenvolvemento de vangarda escollaInstalación Avanzada e selecciona qgis-dev-full

Instaladores autónomos dende paquetes OSGeo4W
Versión máis recente (máis rica en funcionalidades):
  • Instalador Autónomo de QGIS Versión 3.8 (64 bit)
  • sha256
  • Instalador Autónomo de QGIS Versión 3.8 (32 bit)
  • sha256
Repositorio da versión de longo prazo (máis estable):
  • Instalador Autónomo de QGIS Versión 3.4 (64 bit)
  • sha256
  • Instalador Autónomo de QGIS Versión 3.4 (32 bit)
  • sha256

Descarga para Mac OS X

Official All-in-one, signed installers

Mac Installer Packages for macOS High Sierra (10.13) and newer.

Versión máis recente (máis rica en funcionalidades):
  • Versión do Instalador QGIS macOs 3.8
Versión de longo prazo (máis estable):
  • Versión do Instalador QGIS macOs 3.4

Alternative build

Paquetes Instaladores Mac para macOS El Capitán (10.11) e máis novos.

As instrucións de instalación están lo ficheiro Ler na imaxe do disco. Estes paquetes utilizan a python.org Python 3, versión 3.6, a compilación "macosx10.9" - outras distribucións non se admiten. Instala Python antes de instalar QGIS.

Hai dispoñibles plugins de formato GDAL e grades PROJ adicionais en kyngchaos.com.

Descarga para Linux

  • Debian/Ubuntu
  • Fedora
  • openSUSE
  • RHEL, CentOS, Scientific Linux, ...

  • Slackware
