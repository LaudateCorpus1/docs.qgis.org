QGIS telepítők¶

Hogyan szerezhetem be a QGIS-t?

A QGIS szabadon elérhető Windowson, Linuxon, MacOS X-en és Androidon.

Javasoljuk a kiadott szoftver változat csomagjainak telepítését.

A következő kiadás kipróbálásához teszt csomagokat kínálunk, ez lehetővé teszi, hogy a felhasználók is hozzájáruljanak a fejlesztéshez (olvassa el a figyelmeztetést).

A kiadást megelőző, a funkció bővítés befagyasztása idején (lásd Kiadás menetrend) a csomagok előzetes kiadások, melyek tesztelését nyomatékosan javasoljuk a felhasználóknak.

Windows¶

A windowsos telepítőkért menjen a Fő telepítők oldalra.

Az új felhasználóknak az önálló telepítőt ajánljuk.

A tapasztaltabb QGIS felhasználók használhatják az OSGeo4W csomagot, mely lehetővé teszi több változat párhuzamos telepítését.

Az OSGeo4W tárház sok szoftvert tartalmaz az OSGeo projektek közül. A QGIS-t és az összes függőségeket tartalmazza, Python, GRASS, GDAL, stb. A telepítő az internetről képes telepíteni vagy előzetesen letölthetjük az összes szükséges csomagot. A letöltött fájlok egy helyi könyvtárba kerülnek a későbbi telepítéshez. A lépések:

  • Download OSGeo4W Installer (32 bit or 64 bit) and start it

  • Válassza az Advanced Install opciót,

  • navigáljon a Desktop szakaszhoz

  • és válasszon egyet vagy többet a következő csomagokból:

    Kiadás Verzió Csomag Leírás

Azoknak a felhasználóknak akik valamilyen ok miatt nem tudják az OSGeo4W-t használni vagy mást részesítenek előnyben, van egy önálló telepítő a qgis-dev csomaghoz, ami egy heti pillanatkép az OSGeo4W telepítőből (hétfőnként készül). A funkció bővítés befagyasztása idején ez mint kiadásra megjelölt is szerepel.

Figyelmeztetés.

Megjegyzés

Ne telepítse vakon az egész OSGeo4W-t! Elegendő a QGIS-t és néhány szükséges összetevőt, amit szeretne. A függőségek automatikusan hozzá lesznek adva. A teljes telepítés olyan összetevőket hozhat, amik harmadik féltől származó kiegészítők manuális telepítését igénylik. Ezek az összetevők a szükséges kiegészítők nélkül instabillá tehetik a telepített rendszert.

Linux¶

A legtöbb Linux disztribúció A QGIS-t több csomagra bontja; valószínűleg a qgis és a qgis-python (modulok futtatásához) csomagra lesz szüksége. Az olyan csomagokat mint qgis-grass (vagy qgis-plugin-grass), qgis-server először kihagyhatja, vagy akkor installálhatja amikor szüksége lesz rájuk.

Itt talál disztribúciónkénti leírást a telepítésről. A legtöbb disztribúcióhoz létezik telepítési leírás a QGIS stabil és a QGIS legújabb testing változatához (vegye tekintetbe a figyelmeztetést).

Debian/Ubuntu¶

Az alapértelmezett Debian és Ubuntu szoftvertárolók gyakran régebbi QGIS verziót tartalmaznak.

Egy újabb verzió beszerzéséhez, alternatív szoftvertárolót kell megadnia az alábbi deb-lines-ok közül az egyiket adja a /etc/apt/sources.list fájljához.

Fő tárolónk több csomagot tartalmaz számos Debian és Ubuntu verzióhoz, az egyes disztribúciók által szolgáltatott függőségek alapján.

Az Ubuntuhoz külön tárolóban ubuntugis alapú extra csomagokkal rendelkezünk, ami más GIS csomagok aktuálisabb verzióit tartalmazza, mint az Ubuntu maga. Ha azokat is akarja használni, akkor az ubuntugis-unstable ppa-t be kell tennie az /etc/apt/sources.list fájljába (lásd: ubuntugis documentation).

Megjegyzés

A kiadási csomagokat csak nem sokkal egy új verzió kiadása után hozzák létre. Az instabil, még nem kiadott debian verziók (testing) és ubuntugis-unstable könyvtárak megváltozásával előfordulhat, hogy a célcsomagok előbb-utóbb elromlanak, amikor a debian, az ubuntu vagy az ubuntugis-unstable fejlesztése folytatódik, és a csomagok függőségekként vannak jelen qgis módosításában. Ebben az esetben a következőket teheti:

  • áttér a stabil debian verzóra, és az erre a célra kiadott csomagokat használja,
  • megvárja a következő pont kiadást (esedékes: 2019-09-13 12:00:00 UTC), ami minden hónapban megtörténik és tartalmazni fogja a frissített függőségeket,
  • áttér az éjszakai verzió tárolóra (elérhető a két kiadási és mester fejlesztési ágra), amelynek csomagjai rendszeresen újraépülnek, és automatikusan felveszi a frissített függőségeket, vagy
  • felépíti magának a csomag készletekből (lásd INSTALL).

Adja az egyik tárolóhoz tartozó sorokat a /etc/apt/sources.list fájljához:

deb     *repository* *codename* main
deb-src *repository* *codename* main

Example latest release for Debian unstable:

deb     https://qgis.org/debian unstable main
deb-src https://qgis.org/debian unstable main

Ezután gépelje be az alábbi parancsot a QGIS telepítéséhez:

sudo apt-get update
sudo apt-get install qgis qgis-plugin-grass

In case you would like to install QGIS Server, type:

sudo apt-get update
sudo apt-get install qgis-server

Megjegyzés

Kérjük távolítsa el a frissítés előtt az összes QGIS és GRASS csomagot, melyet más tárolókból telepített.

Kulcsszerver hiba esetén qgis.org tároló publikus kulcsát az apt kulcstárolóhoz így lehet hozzáadni:

wget -O - https://qgis.org/downloads/qgis-2019.gpg.key | gpg --import
gpg --fingerprint 51F523511C7028C3

Ezt írja ki:

pub   rsa4096 2019-08-08 [SCEA] [expires: 2020-08-08]
      8D5A 5B20 3548 E500 4487  DD19 51F5 2351 1C70 28C3
uid           [unknown] QGIS Archive Automatic Signing Key (2019) <qgis-developer@lists.osgeo.org>

A kulcsellenőrzés után hozzáadható az apthoz:

gpg --export --armor 51F523511C7028C3 | sudo apt-key add -

Vagy letöltheti a kulcsot egy kulcskiszolgálóról, és hozzáadhatja az apthoz egy lépésben (kézi ellenőrzés nélkül):

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-key 51F523511C7028C3

Fedora¶

A csomag letöltéséhez bármely Fedora verzióhoz gépelje be:

sudo dnf install qgis python3-qgis qgis-grass qgis-server

Az alapértelmezett Fedora szoftvertárolók gyakran régebbi QGIS verziót tartalmaznak.

Egy újabb kiadás beszerzéséhez, alternatív szoftvertárolót kell megadnia a telepíteni kívánt verzió alapján (stable, LTR vagy testing).

Stabil QGIS¶

A tároló engedélyezése:

sudo dnf copr enable dani/qgis

Ezután gépelje be az alábbi parancsot a QGIS telepítéséhez:

sudo dnf install qgis python3-qgis qgis-grass

Abban az esetben, ha telepíteni szeretné a QGIS Servert (vegye figyelembe, hogy nem szokásos gyakorlat az ügyfél- és kiszolgáló alkalmazások telepítése ugyanarra a gépre), írja be:

sudo dnf install qgis-server python3-qgis

This repository also provides a copy of SAGA 2.3.1 compatible with Processing. It can be installed with the following commands:

sudo dnf install saga

ArchLinux¶

Stabil QGIS¶

Archlinux is available in official repository : https://www.archlinux.org/packages/community/x86_64/qgis/

Install with :

pacman -S qgis

QGIS LTR¶

Qgis Long Term Release is available in AUR (Arch User Repository).

Install with yaourt or other package manager which support AUR :

yaourt -S qgis-ltr

For bugs and other behaviour, read comments here : https://aur.archlinux.org/packages/qgis-ltr/

QGIS tesztelés¶

Qgis testing is available in AUR (Arch User Repository).

Install with yaourt or other package manager which support AUR :

yaourt -S qgis-git

For bugs and other behaviour, read comments here : https://aur.archlinux.org/packages/qgis-git

Mac OS X / macOS¶

A telepítési útmutatások ReadMe-ben találhatóak a lemezképen. A letöltések pedig a QGIS letöltések weboldalán vannak.

Aktuális QGIS¶

Az aktuális QGIS csomagok a python.org Python 3.6 verziót, de legalább a 3.6.5-t használják, a „macosx10.9” kiadás - más disztribúciók nincsenek támogatva. A csomagokat a számozott sorrendben telepítse. Különösen jegyezze meg, hogy a Python-t a GDAL Complete és a QGIS csomag előtt telepítse, különben a GDAL és más szükséges Python modul nem lesz telepítve.

Stabil QGIS¶

A stabil csomag a Python 2.7 rendszert használja - más disztribúciók nem támogatottak. Szükség van továbbá a NumPy és a Matplotlibcsomagokra a lemezképről. A csomagokat a számozott sorrendben telepítse. Ha egy régebbi fő verzióról (2.16 vagy régebbi) frissít, akkor a telepítés előtt törölje a QGIS.app-ot az Alkalmazásokból.

További Python modulok érhetőek el itt: kyngchaos.com.

Standalone Installer¶

A standalone installer is available at https://lutraconsulting.github.io/qgis-mac-packager/.

FreeBSD¶

Stabil QGIS¶

To install QGIS from binary packages type

pkg install qgis

Or to customize compilation options, you can build it from FreeBSD ports

cd /usr/ports/graphics/qgis
make install clean

QGIS LTR¶

To install QGIS from binary packages type

pkg install qgis-ltr

Or to customize compilation options, you can build it from FreeBSD ports

cd /usr/ports/graphics/qgis-ltr
make install clean

Flatpak¶

There is an experimental QGIS flatpak for QGIS Stable available.

For general Linux Flatpak install notes, see https://flatpak.org/setup/

QGIS on Flathub: https://flathub.org/apps/details/org.qgis.qgis

To install:

flatpak install --from  https://flathub.org/repo/appstream/org.qgis.qgis.flatpakref

Then to run:

flatpak run org.qgis.qgis

To update your flatpak QGIS:

flatpak update

Flathub files: https://github.com/flathub/org.qgis.qgis

Android¶

Egy kísérleti változat érhető el a google play store-ban.

https://play.google.com/store/apps/details?id=org.qgis.qgis

Figyelem

Jelenleg nincs az Android 5-höz támogatás. A legjobb támogatás az Android 4.3 és 4.4.x verziókhoz érhető el. Ez egy közvetlen portja a QGIS asztali alkalmazásnak. Csak enyhén optimalizálták az érintő képernyőkhöz, ezért a mindennapos használata gondos mérlegelést igényel. Vannak más alkalmazások, melyeket érintőképernyőre terveztek és optimalizáltak.

QGIS teszt figyelmeztetés¶

Figyelem

QGIS teszt csomagot is készítünk néhány platformra a QGIS stabil változata mellett. A QGIS teszt még nem kiadott szoftvert tartalmaz, melyen most is dolgoznak. Ezek csak tesztelési célokra szolgálnak a korai felhasználók számára, hogy ellenőrizzék a hibák kijavítását és hogy újabb hiba nem került be. Bár gondosan próbáljuk elkerülni az összeomlásokat, bármelyik pillanatban nem működhet vagy az adataival rossz dolgok történhetnek. Vigyázzon! Mi figyelmeztettük!
