Document QGIS

The QGIS project aims to provide clear documentation concerning a User guide, GIS concepts, and training material. Writing documentation provides an ideal platform for your first contribution to the project. If you want to join to the documentation effort, you can find a gentle introduction and guidelines here:

Write Documentation

QGIS fordítás

A QGIS projekt egy egyszerű GIS bevezetőt, egy felhasználói kézikönyvet, egy kódolási és fordítási kézikönyvet és egy API dokumentációt kínál angol nyelven, valamint néhány esetben számos idegen nyelvre lefordított változatot. Amennyiben érdekelt abban, hogy részt vegyen a dokumentációs erőfeszítésekben, pl. angol kézikönyv fordítása más nyelvre, itt megtanulhatja, hogy hogyan induljon el:

QGIS lefordítása az Ön nyelvére

QGIS mag fejlesztés

A QGIS mag egyrészt könyvtárakból áll, melyek lehetővé teszik, hogy testreszabott alkalmazást hozzon létre a hatékony API segítségével. Másrészt pedig asztali és szerver alkalmazásból áll, melyek egy felhasználóbarát felületet biztosítva teszik elérhetővé a könyvtárak által biztosított lehetőségeket.
