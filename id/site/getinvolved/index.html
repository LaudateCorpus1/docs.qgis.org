Get Involved in the QGIS Community

QGIS is developed by a team of dedicated volunteers and organisations. We strive to be a welcoming community for people of all race, creed, gender and walks of life.

Anda dapat turut berperan!

For more information, please read our pernyataan keragaman dan kode etik.

Laporan Kesalahan

If you find an issue working with QGIS, please

report it

so developers can check and fix it.

Berpartisipasi dalam Saluran Dukungan

There are several

  • QGIS mailing lists,
  • IRC chats,
  • forums, dan
  • user groups
you can use to get help from fellow users and developers, or to keep up with current project events. You are welcome to subscribe to any of the lists. Please remember to contribute to the list by answering questions and sharing your experiences. We also provide a website to search the mailing list archives. For the full list see

our community support channels page

Document QGIS

The QGIS project aims to provide clear documentation concerning a User guide, GIS concepts, and training material. Writing documentation provides an ideal platform for your first contribution to the project. If you want to join to the documentation effort, you can find a gentle introduction and guidelines here:

Terjemahkan QGIS

Proyek QGIS menyediakan pengenalan awal GIS, Panduan Pengguna, Panduan Coding dan Kompilasi serta Dokumentasi API dalam bahasa Inggris yang pada beberapa kasus juga diterjemahkan dalam berbagai bahasa asing. Apabila anda tertarik untuk turut bergabung dengan usaha penerjemahan, seperti menterjemahkan manual bahasa Inggris dalam bahasa lain, pelajari lebih dulu bagaimana melakukannya di sini:

Terjemahkan QGIS dalam bahasa anda

Pengembangan Inti QGIS

QGIS Core terdiri dari sekumpulan pustaka yang memungkinkan anda untuk membuat aplikasi sendiri dengan menyediakan API yang ampuh di satu sisi, Di sisi yang lain ia terdiri dari aplikasi desktop dan server yang mengungkap kemungkinan yang ditawarkan oleh pustaka, memungkinkan suatu antarmuka yang user friendly.
