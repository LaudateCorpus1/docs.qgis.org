自分の環境にあったQGISのダウンロード

バイナリパッケージ（インストーラ）はこのページから入手できます。

最新バージョンは QGIS 3.8.2 'Zanzibar' で、 16.08.2019にリリースされました。

現在提供されている長期リポジトリは QGIS 3.4.11 'Madeira' です.　

QGISは Windows・MacOSX・Linux・Android版がリリースされています。

Windows版のダウンロード

OSGeo4WのQGIS：
  • OSGeo4W ネットワークインストーラー (64 bit)
  • OSGeo4W ネットワークインストーラー (32 bit)

インストーラの中で次の項目を選択してくださいエクスプレスデスクトップインストール そして QGIS インストールする 最新のリリース.
取得するには 長期リリース （これは最新リリースではない）選択して下さい上級インストール そして選択 qgis-ltr-full
取得するには 最先端の開発ビルド 選択する 上級インストール そして選択 qgis-dev-full

Alternative build

El Capitan (10.11) 以上のMacOS用インストーラパッケージです。

インストール手順はディスクイメージのRead Meにあります。これらのパッケージは python.org Python 3, 3.6版の"macosx 10.9"ビルドを使います。他のディストリビューションには対応していません。QGISをインストールする前にPythonをインストールして下さい。

追加のGDALフォーマットのプラグインとPROJグリッドは次のサイトにあります。 kyngchaos.com.

Linux版のダウンロード

GNU/Linux の様々なディストリビューション向けのバイナリパッケージ (rpm や deb) や、(インストールマネージャ向けの) ソフトウェアリポジトリが利用可能です。以下のディストリビューションから選択してください：

  • Debian/Ubuntu
  • Fedora
  • openSUSE
  • RHEL, CentOS, Scientific Linux, ...
  • Mandriva
  • Slackware
  • ArchLinux
  • Flatpak
