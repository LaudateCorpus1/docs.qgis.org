ឯកសារQGIS

QGIS has a lot of documentation. All documentation is in English but some documents such as the user guide are also available in other languages.

You will find documentation for every QGIS long term release on the respective documentation website.

Get involved and help us write a better documentation.

DOCUMENTATION

  • QGIS 3.4
  • QGIS testing (>3.4)
  • Documentation for older versions

QGIS 3.4

For users:

  • Read the QGIS User guide (or as PDF)
  • Follow the tutorials in the QGIS Training manual (or as PDF)
  • Learn GIS basics in A gentle introduction in GIS

For documentation writers:

  • Learn how to write docs using the Documentation Guidelines

For developers:

  • Go Python, study the PyQGIS cookbook (for plugins and scripting) (or as PDF)
  • C++ Api documentation
  • PyQGIS - QGIS Python Api documentation

QGIS testing

We are still updating (not translating yet) the documentation for releases newer than QGIS 3.4. We call this version 'QGIS testing' and can be found here: http://docs.qgis.org/testing
