QGIS Dokumentacija

QGIS turi daug dokumentacijos. Visa dokumentacija yra angliška, bet kai kurie dokumentai, tokie kaip naudotojo vadovas, yra išversti ir į kitas kalbas.

You will find documentation for every QGIS long term release on the respective documentation website.

Get involved and help us write a better documentation.
