Download QGIS voor jouw platform

Binaire pakketten (installatieprogramma's) zijn beschikbaar vanaf deze pagina.

De huidige versie is QGIS 3.8.2 'Zanzibar' en werd uitgegeven op 16.08.2019.

De opslagplaatsen voor lange-termijn bieden momenteel QGIS 3.4.11 'Madeira' aan.

QGIS is beschikbaar voor Windows, MacOS X, Linux en Android.

Download voor Windows

QGIS in OSGeo4W:
  • OSGeo4W Netwerk Installatiepakket (64 bit)
  • OSGeo4W Netwerk Installatiepakket (32 bit)

Tijdens de installatie kies je Desktop Express Install en selecteer QGIS om de laatste uitgave te installeren.
Kies, om de lange-termijn uitgave (dat is niet ook de laatste uitgave) te verkrijgen Geavanceerde installatie en selecteer qgis-ltr-full
Kies, om de allernieuwste ontwikkelversie te verkrijgen Geavanceerde installatie en selecteer qgis-dev-full

Zelfstandige installatieprogramma's uit pakketten van OSGeo4W
Laatste uitgave (het rijkst aan mogelijkheden):
  • QGIS standalone installatiepakket Versie 3.8 (64 bit)
  • sha256
  • QGIS standalone installatiepakket Versie 3.8 (32 bit)
  • sha256
Lange-termijn uitgave opslagplaats (meest stabiel):
  • QGIS standalone installatiepakket Versie 3.4 (64 bit)
  • sha256
  • QGIS standalone installatiepakket Versie 3.4 (32 bit)
  • sha256

Download voor Mac OS X

Official All-in-one, signed installers

Mac Installer Packages for macOS High Sierra (10.13) and newer.

Laatste uitgave (het rijkst aan mogelijkheden):
  • QGIS macOS Installatiepakket versie 3.8
Lange-termijn uitgave (meest stabiel):
  • QGIS macOS Installatiepakket versie 3.4

Alternative build

Mac Installer Packages voor macOS El Capitan (10.11) en nieuwer.

Instructies voor installeren staan in de ReadMe op de schijf. Deze pakketten gebruiken de python.org Python 3, versie 3.6, de "macosx10.9" build - andere distributies worden niet ondersteund. Installeer Python vóór het installeren van QGIS.

Aanvullende plug-ins in de indeling GDAL en rasters voor PROJ zijn beschikbaar op kyngchaos.com.
