QGIS — um instrumento interessante para o planeamento florestal conceitos a nível local e regional¶

Quase 35% ‘% o’ da superfície da Suíça é coberto por florestas. Novas exigências sobre a floresta e a má situação financeira das empresas florestais na Suíça, levou ao desenvolvimento de um novo conceito de planeamento florestal à cerca de quinze anos atrás. Este conceito introduziu uma distinção clara entre o planeamento ao nível das autoridades e do planeamento ao nível do proprietários. Como novo escritório de planeamento florestal, o Guaraci trabalha principalmente na região de Basel (Suíça), em ambos os níveis de planeamento mencionados. Este caso de estudo discute o uso do QGIS ao nível do proprietário florestal. A este nível, um sistema de informação geográfica simples, dinâmica e compatível é necessário para o estabelecimento de um plano de gestão florestal. Este plano abrange cerca de 1000 ha de florestas, será válido para cerca de 20 anos e é implementado pelo gestor florestal local. A área está situada na região de Basileia (ver google maps).

Como o QGIS é usado¶

Dentro do planeamento de todo o processo, o QGIS foi utilizado para as seguintes quatro tarefas:

  • Importação de dados existentes. Os dados incluem mapas de fundo geo-referenciados (*.tif), juntamente com os arquivos do tipo polígono, linha e ponto. Para o gestor da floresta, a visualização de dados temáticos existentes (por exemplo, os recursos hídricos, solo e água potável, zonas de protecção da natureza) são importantes para o seu trabalho diário.
  • Digitalização. As medições florestais previstas pelo gestor florestal foram digitalizadas. As ferramentas de digitalização do QGIS com as opções avançadas de atracção são muito úteis e amigáveis. O módulo de “camadas de texto delimitado” permitiu a importação de tabelas de dados adicionais.
  • Análise. Foram calculadas as áreas das diferentes categorias de medição. A partir de certas características da linha, como trilhos de comboio ou estradas, as florestas de protecção foram determinadas com a ajuda de uma área envolvente dessas linhas.
  • Impressão e exportação. O compositor de mapas suporta criação de um layout para o mapa. O mapa foi exportado em ficheiro PDF e imagem.

Conclusão¶

QGIS é uma ferramenta interessante para pequenos escritórios de planeamento, como o Guaraci, bem como para os gestores florestais e proprietários. Ele pode ser usado num contexto profissional e o resultado de saída apresentada foi apreciada por todas as partes envolvidas. Para as tarefas mencionadas na presente versão QGIS “1,4 Enceladus”, oferece todas as ferramentas SIG necessárias. As qualidades mais importantes do QGIS são a interface de usuário intuitiva e compatibilidade com diferentes formatos de arquivos. É uma ferramenta ideal para projetos que, por um lado devido aos dados geográficos existentes e, por outro lado, ser necessário para a aquisição de novos dados. O manual é útil e fácil de entender. Tanto em termos técnicos, bem como em perspectivas financeiras o QGIS é uma alternativa interessante comparado com o outro software SIG.

Perspectiva¶

As tarefas principais do planeamento florestal podem ser trabalhadas com as ferramentas básicas SIG mencionadas. O Guaraci vai continuar a usar QGIS para o planeamento florestal, a nível do proprietário florestal. Com base nas experiências com o projecto apresentado, o QGIS também pode ser recomendado para conceitos florestais ao nível das autoridades locais. Projectos maiores e mais complexos, com requisitos de análise detalhadas poderiam ser tratadas com a combinação do QGIS com o servidor de base de dados espaciais Postgis. Novas zonas de madeira, bem como florestas reservadas e zonas de protecção podem ser facilmente digitalizadas. Felizmente, os proprietários e gestores florestais utilizarão cada vez mais o potencial do QGIS para seu planeamento anual, bem como a documentação das suas actividades florestais.

Raphael Häner
