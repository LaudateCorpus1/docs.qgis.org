Material de formação¶

Nesta página, pode ser listado se fornecer material de formação GRATUITA para o QGIS.

A ideia geral, é podermos (re)utilizar materiais uns dos outros.

Os educadores são encorajados a utilizar/misturar/adicionar material ao QGIS - Manual de Formação.

Please write to psc@qgis.org if you want to have something added here. Or you edit this page on Github and do a pull request.

Inglês¶

O Manual de Formação oficial pode ser encontrado aqui: http://docs.qgis.org/latest/en/docs/training_manual/index.html

Material de tutoriais, originalmente desenvolvidos como parte de um workshop para Ecologia e o Programa de Biologia Evolucionária na Universidade A&M do Texas (produzido e gerido pelo Mike Treglia): http://mltconsecol.github.io/QGIS-Tutorial/

O material de formação em pessoa, originalmente desenvolvido para o Departamento de Transportes da cidade de Nova Iorque por Datapolitan <http://www.datapolitan.com> _:

  • Introdução aos Fundamentos do GIS <http://training.datapolitan.com/qgis-training/Introduction_to_GIS_Fundamentals> _
  • GIS Intermédio com QGIS e PostGIS <http://training.datapolitan.com/qgis-training/Intermediate_GIS/> _
  • Código-fonte para materiais de formação <https://github.com/Datapolitan-Training/qgis-training> _

Português do Brasil¶

Roberto Ilacqua escreveu um manual para classificação de imagens no QGIS: Manual do QGIS para Classificação Supervisionada de Áreas <https://malariaeliminationblog.files.wordpress.com/2017/11/manual-qgis.pdf> _, versão completa, 200 p - 2017 ou Manual do QGIS para a Classificação Supervisionada de Áreas (versão curta) <https://malariaeliminationblog.files.wordpress.com/2018/01/protocol_lulc_methods.pdf> _, versão curta, 34 p - 2017.

Francês¶

O Ministério Francês do Ambiente suporta o projeto QGIS. O material de formação usado está disponível aqui: http://www.geoinformations.developpement-durable.gouv.fr/qgis-supports-pedagogiques-r947.html

Outros recursos estão disponíveis aqui: http://www.geoinformations.developpement-durable.gouv.fr/qgis-r625.html

O Centro Nacional Francês de Pesquisa Científica (CNRS) criou um tutorial on-line do QGIS: http://ouvrir.passages.cnrs.fr/tutoqgis/

Alemão¶

Material HSR: http://giswiki.hsr.ch/QGIS-Materialien

Checo¶

QGIS para iniciantes: http://training.gismentors.eu/qgis-zacatecnik/

QGIS para utilizadores avançados: http://training.gismentors.eu/qgis-pokrocily/
