Documentație QGIS

QGIS dispune de foarte multă documentație. Majoritatea documentelor sunt în limba engleză, dar unele materiale, cum ar fi ghidul de utilizare, sunt disponibile și în alte limbi.

You will find documentation for every QGIS long term release on the respective documentation website.

Get involved and help us write a better documentation.
