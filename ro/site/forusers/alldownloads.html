Pachete de instalare QGIS¶

Cum puteți obține QGIS?

QGIS este disponibil gratuit pentru Windows, Linux, MacOS X, BSD și Android.

Vă recomandăm instalarea pachetelor software oficiale.

De asemenea, pentru evaluarea viitoarei versiuni oficiale, și pentru a permite non-dezvoltatorilor să sprijine dezvoltarea, oferim pachete de testare (a se vedea avertismentul).

În faza de înghețare, care precede o versiune oficială (v. Planificarea versiunilor), aceste pachete sunt efectiv pre-lansate, îndemnând utilizatorii să le testeze.

Windows¶

Pentru instalarea pe Windows mergeți la Pagina principalelor kit-uri de instalare.

Pentru noii utilizatori se recomandă kit-urile de instalare offline.

Utilizatorii QGIS avansați pot folosi pachetul OSGeo4W, cu ajutorul căruia e posibilă instalarea mai multor versiuni în paralel.

Depozitul OSGeo4W conține o mulțime de software din proiectele OSGeo. QGIS și toate dependențele sunt incluse, alături de Python, GRASS, GDAL, etc. Aplicația permite efectuarea instalării direct de pe internet sau doar descărcarea prealabilă a tuturor pachetelor necesare. Fișierele descărcate sunt păstrate într-un director local, pentru instalarea ulterioară. Pașii necesari sunt:

  • Download OSGeo4W Installer (32 bit or 64 bit) and start it

  • Selectați Instalare avansată,

  • mergeți în secțiunea Desktop și

  • alegeți unul sau mai multe dintre următoarele pachete:

    qgis-rel-dev [1] Compilația de noapte a viitoarei versiuni minore

Pentru utilizatorii care, din diverse rațiuni, nu pot folosi OSGeo4W, există, de asemenea, o versiune săptămânală de qgis-dev de la OSGeo4W (creată în ziua de luni), sub formă de instalator independent. Totodată, în faza de înghețare are rol de versiune candidat.

Rețineți avertismentul.

Notă

Nu efectuați orbește o instalare OSGeo4W completă. Instalați doar QGIS și poate alte componente care vă plac. Dependențele vor fi incluse în mod automat. O instalare completă aduce componente care necesită instalarea manuală a unor terțe părți. Aceste componente fac instalarea inutilizabilă, în lipsa acestor adaosuri.

Linux¶

Cele mai multe distribuții Linux împart QGIS în mai multe pachete; probabil veți avea nevoie de qgis și qgis-python (pentru a rula plugin-urile). Pachetele precum qgis-grass (sau qgis-plugin-grass) și qgis-server pot fi ignorate inițial, sau pot fi instalate numai atunci când aveți nevoie de ele.

In continuare veți găsi instrucțiuni pentru instalarea specifică fiecărei distribuții. Majoritatea distribuțiilor dispun de instrucțiuni de instalare atât pentru QGIS stabil cât și pentru un QGIS de ultimă oră, pentru testare (rețineți avertismentul).

Debian/Ubuntu¶

Depozitele software implicite ale Debian și Ubuntu, adesea, conțin versiuni mai vechi de QGIS.

Pentru a obține versiunile noi, trebuie să apelați la depozite software alternative, prin adăugarea uneia din liniile deb de mai jos, în fișierul /etc/apt/sources.list.

Depozitul nostru principal conține multiple linii către diverse versiuni de Debian și Ubuntu, în funcție de dependențele existente în distribuțiile individuale.

For Ubuntu we also have extra packages in a separate repository that are based on ubuntugis, which holds more uptodate versions of other GIS packages than Ubuntu itself. If you want those you also need to include ubuntugis-unstable ppa in your /etc/apt/sources.list file (see ubuntugis documentation).

Notă

The release packages are only produced once shortly after a new version has been released. As unstable, not yet released debian versions (testing) and ubuntugis-unstable can have library changes the packages might sooner or later be broken for these targets, when the development in debian, ubuntu or ubuntugis-unstable moves on and their packages used as dependencies in qgis change. In that event you can either

  • să treceți la o versiune Debian stabilă și să folosiți pachetele oficiale,
  • să așteptați lansarea următoarei versiuni minore (conform 2019-09-13 12:00:00 UTC), care are loc în fiecare lună și include dependențele actualizate,
  • să alegeți un depozit de noapte (disponibil pentru două versiuni, oficială și master), ale căror pachete sunt reconstruite cu regularitate și ale căror dependențe sunt actualizate în mod automat, sau
  • să vă construiți propriul set de pachete (vedeți INSTALL).

Adăugați liniile specifice unuia dintre depozite, la fișierul dvs. /etc/apt/sources.list:

deb     *repository* *codename* main
deb-src *repository* *codename* main

Example latest release for Debian unstable:

deb     https://qgis.org/debian unstable main
deb-src https://qgis.org/debian unstable main

Apoi, introduceți comenzile de mai jos, pentru a instala QGIS:

sudo apt-get update
sudo apt-get install qgis qgis-plugin-grass

In case you would like to install QGIS Server, type:

sudo apt-get update
sudo apt-get install qgis-server

Notă

Înainte de a face actualizarea, eliminați toate pachetele QGIS și GRASS pe care este posibil să le aveți instalate din alte depozite,.

În caz de erori ale serverului de chei adăugați cheia publică a depozitului qgis.org la inelul de chei apt, tastând:

wget -O - https://qgis.org/downloads/qgis-2019.gpg.key | gpg --import
gpg --fingerprint 51F523511C7028C3

Rezultatul așteptat:

pub   rsa4096 2019-08-08 [SCEA] [expires: 2020-08-08]
      8D5A 5B20 3548 E500 4487  DD19 51F5 2351 1C70 28C3
uid           [unknown] QGIS Archive Automatic Signing Key (2019) <qgis-developer@lists.osgeo.org>

După ce ați verificat amprenta, puteți adăuga cheia apt cu:

gpg --export --armor 51F523511C7028C3 | sudo apt-key add -

Alternativ, puteți descărca cheia de la un keyserver și să o adăugați în apt într-un singur pas (fără verificarea manuală a amprentelor digitale)

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-key 51F523511C7028C3

Fedora¶

Obțineți pachete pentru oricare versiune de Fedora tastând:

sudo dnf install qgis python3-qgis qgis-grass qgis-server

Default Fedora software repositories often hold older versions of QGIS.

To have newer versions, you have to add alternative software repositories based on the version you want to install (stable, LTR or testing).

QGIS stabil¶

Enable the repository:

sudo dnf copr enable dani/qgis

Apoi, introduceți comenzile de mai jos, pentru a instala QGIS:

sudo dnf install qgis python3-qgis qgis-grass

În cazul în care doriți să instalați QGIS Server (rețineți că instalarea pe aceeași mașină a aplicațiilor client și server nu reprezintă o practică obișnuită), tastați:

sudo dnf install qgis-server python3-qgis

This repository also provides a copy of SAGA 2.3.1 compatible with Processing. It can be installed with the following commands:

sudo dnf install saga

openSUSE¶

QGIS stabil¶

Cel mai recent pachet stabil de qgis pentru openSUSE este disponibil pentru 13.1, 13.2, Leap_42.1, Leap_42.2 și Tumbleweed (32 și 64 de biți). Adăugați următorul depozit în managerul dvs. de instalare, unde <VERSION> reprezintă, de exemplu, «openSUSE_Tumbleweed».

https://download.opensuse.org/repositories/Application:/Geo/<VERSION>/

Toate pachetele includ suport GRASS și Python.

QGIS LTR (Versiune cu Suport pe Termen Lung)¶

Cel mai recent pachet qgis-ltr, cu Suport pe Termen Lung, pentru openSUSE este disponibil pentru 13.1, 13.2, Leap_42.1, Leap_42.2 și Tumbleweed (32 și 64 de biți). Adăugați următorul depozit în managerul dvs. de instalare, unde <VERSION> reprezintă, de exemplu, «openSUSE_Tumbleweed».

https://download.opensuse.org/repositories/Application:/Geo/<VERSION>/

Toate pachetele includ suport GRASS și Python.

QGIS pentru testare¶

Un pachet qgis-master, de dezvoltare, actualizatcu periodicitate, este disponibil pentru 13.1, 13.2, Leap_42.1, Leap_42.2 și Tumbleweed (32 și 64 biți). Adăugați următorul depozit în managerul dvs. de instalare, unde <VERSION> reprezintă, de exemplu, «openSUSE_Tumbleweed».

https://download.opensuse.org/repositories/Application:/Geo/<VERSION>/

ArchLinux¶

QGIS stabil¶

Archlinux is available in official repository : https://www.archlinux.org/packages/community/x86_64/qgis/

Install with :

pacman -S qgis

QGIS LTR¶

Qgis Long Term Release is available in AUR (Arch User Repository).

Install with yaourt or other package manager which support AUR :

yaourt -S qgis-ltr

For bugs and other behaviour, read comments here : https://aur.archlinux.org/packages/qgis-ltr/

QGIS pentru testare¶

Qgis testing is available in AUR (Arch User Repository).

Install with yaourt or other package manager which support AUR :

yaourt -S qgis-git

For bugs and other behaviour, read comments here : https://aur.archlinux.org/packages/qgis-git

Mac OS X / macOS¶

Installation instructions are in the ReadMe on the disk image. Downloads are on the QGIS download page.

Versiunea QGIS actuală¶

The current QGIS package uses the python.org Python 3.6, at least version 3.6.5, the „macosx10.9” build - other distributions are not supported. Install packages in the numbered order. Especially note that Python must be installed before the GDAL Complete package and QGIS, else the GDAL and other needed Python modules will not be installed.

QGIS stabil¶

Pachetul stabil utilizează sistemul Python 2.7 - alte distribuții nefiind acceptate. De asemenea, sunt necesare pachetele NumPy și Matplotlib din imaginea discului. Instalați pachetele în ordinea numerotată. Dacă actualizați o versiune majoră veche (2,16 sau anterioară), mai întâi ștergeți QGIS.app din folderul Aplicațiilor.

Alte module Python dedicate pluginurilor sunt disponibile la kyngchaos.com.

Standalone Installer¶

A standalone installer is available at https://lutraconsulting.github.io/qgis-mac-packager/.

FreeBSD¶

QGIS stabil¶

To install QGIS from binary packages type

pkg install qgis

Or to customize compilation options, you can build it from FreeBSD ports

cd /usr/ports/graphics/qgis
make install clean

QGIS LTR¶

To install QGIS from binary packages type

pkg install qgis-ltr

Or to customize compilation options, you can build it from FreeBSD ports

cd /usr/ports/graphics/qgis-ltr
make install clean

Flatpak¶

There is an experimental QGIS flatpak for QGIS Stable available.

For general Linux Flatpak install notes, see https://flatpak.org/setup/

QGIS on Flathub: https://flathub.org/apps/details/org.qgis.qgis

To install:

flatpak install --from  https://flathub.org/repo/appstream/org.qgis.qgis.flatpakref

Then to run:

flatpak run org.qgis.qgis

To update your flatpak QGIS:

flatpak update

Flathub files: https://github.com/flathub/org.qgis.qgis

Android¶

Este disponibilă o nouă versiune experimentală pe google play store.

https://play.google.com/store/apps/details?id=org.qgis.qgis

Atenționare

În prezent nu există suport pentru Android 5. Cel mai bun suport este oferit pentru Android 4.3 și 4.4.x. Aceasta este o portare directă a aplicației QGIS desktop. Ea este doar ușor optimizată pentru dispozitive tactile și, prin urmare, trebuie să fie evaluată cu atenție pentru oportunitatea utilizării de zi cu zi. Există și alte aplicații disponibile, care sunt proiectate și optimizate special pentru dispozitive tactile.

Avertisment de testare QGIS¶

Atenționare

Versiunile de testare QGIS sunt elaborate pentru câteva platforme în plus față de versiunea QGIS stabilă. QGIS de testare conține software nelansat oficial, la care încă mai e de lucru. Aceste versiuni sunt furnizate numai în scop de testare timpurie, pentru a se verifica dacă problemele au fost rezolvate și dacă nu au fost introduse noi erori. Deși lucrăm cu atenție, în orice moment ele ar putea să nu mai funcționeze sau chiar să afecteze într-un mod nedorit datele dumneavoastră. Acționați cu prudență. Vi s-a adus la cunoștință!
