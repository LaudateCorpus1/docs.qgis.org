Descărcați QGIS pentru platforma dumneavoastră

Pachetele binare (kiturile de instalare) se pot descărca de aici.

Versiunea curentă este QGIS 3.8.2 'Zanzibar', care a fost lansată în 16.08.2019.

Depozitele long-term oferă în mod curent QGIS 3.4.11 'Madeira'.

QGIS este disponibil pentru Windows, MacOS X, Linux și Android.

Descărcați pentru Windows

QGIS în OSGeo4W:
  • Instalare online OSGeo4W (64 biți)
  • Instalare online OSGeo4W (32 biți)

La rularea programului de instalare alegeți Desktop Express Install apoi selectați QGIS pentru a instala cea mai nouă versiune.
Pentru a obține versiunea cu suport pe termen îndelungat (care nu este însă și cea mai nouă) alegeți Instalare Avansată apoi selectați qgis-ltr-full
Pentru a obține versiunea cea mai performantă optați pentru Instalare Avansată apoi selectați qgis-dev-full

Official All-in-one, signed installers

  • Kit de instalare QGIS macOS versiunea 3.8

Alternative build

Pachet de Instalare pentru Mac, versiunile macOS El Capitan (10.11) și mai noi.

Instrucțiunile de instalare se află în fișierul ReadMe din imaginea discului. Aceste pachete folosesc python.org Python 3, versiunea 3.6, 'macosx10.9' - alte distribuții nu sunt acceptate. Instalați Python înainte de a instala QGIS.

Mai multe pluginuri pentru formate GDAL și grile PROJ sunt disponibile la kyngchaos.com.

Descărcați pentru Linux

Pentru diversele distribuții de GNU/Linux sunt disponibile pachete binare (rpm și deb) sau depozite de software (pe care le puteți adăuga la propriul manager de instalare). Vă rog să alegeți distribuția preferată, din lista următoare:

  • Debian/Ubuntu
  • Fedora
  • openSUSE
  • RHEL, CentOS, Scientific Linux, ...
  • Mandriva
  • Slackware
  • ArchLinux
  • Flatpak
